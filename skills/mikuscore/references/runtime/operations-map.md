# Operations Map

Use this reference when you need the supported operation list or the preferred document search order.

## Operations

Conversation workflow:

- `convert`: explain source-to-target conversion paths in `mikuscore`
- `diagnostics`: explain warnings, errors, preservation, and loss handling
- `format-guidance`: explain the role and current status of each format in `mikuscore`
- `ai-handoff`: explain why current AI-facing full-score exchange uses `ABC` while canonical score storage remains `MusicXML`
- `workflow`: explain CLI commands, build steps, and test commands
